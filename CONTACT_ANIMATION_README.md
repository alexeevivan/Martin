# Contact Page Animation - Документация

## Что было сделано

### 1. Адаптация React-компонента для Vanilla JS
Файл: `src/js/effects/contact.js`

Преобразован React-компонент с Three.js в обычную JavaScript функцию `createColorBends()`, которая:
- Создает WebGL canvas с shader-анимацией
- Поддерживает интерактивность (реакция на движение мыши)
- Автоматически адаптируется под размер контейнера
- Имеет метод `destroy()` для очистки ресурсов

### 2. Система управления отображением
Файл: `src/js/contactAnimation.js`

Создана система, которая:
- **На десктопе (>768px):**
  - Показывает `.panel-right` с анимацией
  - Скрывает `.card.desktop` и `.accents.desktop`
  
- **На мобильных устройствах (≤768px):**
  - Скрывает `.panel-right`
  - Показывает `.card` и `.accents` (убирает класс `desktop`)

### 3. Автоматическое переключение
- Отслеживает изменение размера окна
- Отслеживает изменение ориентации устройства
- Автоматически пересоздает/уничтожает анимацию при переходе между режимами
- Использует debounce для оптимизации производительности

### 4. CSS медиа-запросы
Файл: `src/styles/contact-card.scss`

Добавлены медиа-запросы для надежного отображения:
```scss
@media screen and (max-width: 768px) {
  .panel-right { display: none !important; }
  .card.desktop, .accents.desktop { display: flex !important; }
}

@media screen and (min-width: 769px) {
  .panel-right { display: flex !important; }
  .card.desktop, .accents.desktop { display: none !important; }
}
```

## Параметры анимации

Анимация настраивается через объект опций:

```javascript
{
  rotation: 45,           // Начальный угол поворота
  speed: 0.15,           // Скорость анимации
  colors: [              // Массив цветов в hex формате
    '#ff6c6c', 
    '#6c9fff', 
    '#ff6cfd', 
    '#6cffa8'
  ],
  transparent: false,    // Прозрачность фона
  autoRotate: 5,        // Скорость автоповорота
  scale: 1.2,           // Масштаб
  frequency: 1.2,       // Частота волн
  warpStrength: 0.8,    // Сила искажения
  mouseInfluence: 0.5,  // Влияние мыши
  parallax: 0.3,        // Эффект параллакса
  noise: 0.1,           // Уровень шума
  iterations: 3,        // Количество итераций
  intensity: 1.8,       // Интенсивность цвета
  bandWidth: 5          // Ширина цветовых полос
}
```

## Инициализация

Анимация автоматически инициализируется при загрузке страницы через:

```javascript
document.addEventListener("DOMContentLoaded", function () {
  setupContactPage();
});
```

## Производительность

- Использует `ResizeObserver` для эффективного отслеживания изменений размера
- Debounce для обработки resize событий (250ms)
- Автоматическая очистка WebGL ресурсов при уничтожении
- Анимация создается только на десктопе, экономя ресурсы на мобильных устройствах

## Совместимость

- Современные браузеры с поддержкой WebGL
- Three.js (уже включен в проект)
- Адаптивный дизайн для всех размеров экранов
- Поддержка touch-устройств

## Отладка

Если анимация не отображается, проверьте:
1. Консоль браузера на наличие ошибок
2. Наличие контейнера `.panel-right .color-bends-container` в DOM
3. Размер окна (анимация работает только на десктопе >768px)
4. Поддержку WebGL в браузере
